# Glossary

- `columbarium`: this repository.
- `BitGN`: benchmark platform at <https://bitgn.com>.
- `benchmark loop`: change agent -> run benchmark -> review score -> iterate.
- `platform layer`: code that talks to benchmark APIs.
- `agent layer`: code that decides actions and reasoning flow.
- `replaceable implementation`: agent logic that can be swapped via adapters/interfaces.
- `experiment record`: at minimum, commit SHA and score.
