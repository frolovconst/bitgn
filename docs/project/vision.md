# Vision

Build a flexible benchmark platform for trying agent implementations and improving BitGN score.

## What success means

- Score improves over time on BitGN.
- Agent logic can be replaced with low friction.
- Experiments are small, isolated, and comparable.
- Each recorded result includes commit SHA and benchmark score.

## Working style

- Prefer measurable benchmark impact.
- Keep interfaces simple and evolving.
- Avoid overengineering.
- Decompose large tasks before implementation.
