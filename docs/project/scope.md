# Scope

## In scope today

- `local-llms/` as an isolated workspace for local LLM experimentation
- reproducible development flows using Nix and checked-in scripts
- workspace-local model storage and operational guidance

## Likely future scope

- additional experimental workspaces
- evaluation tooling
- agent workflows and prompts
- model comparison or benchmarking

## Out of scope until explicitly added

- production infrastructure
- broad product requirements not represented in the repo
- undocumented shared assumptions across future subprojects

## Scope notes

Update this file whenever the repository takes on a materially new responsibility.
