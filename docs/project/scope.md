# Scope

## In scope today

- building and iterating on an AI agent intended to compete on BitGN benchmark challenges
- benchmark-oriented experimentation with prompting, tool use, memory, planning, and reliability
- reproducible local development flows using Nix and checked-in scripts
- local model runtime support when it helps benchmark iteration
- documentation of benchmark assumptions, learnings, and execution plans

## Likely future scope

- benchmark harnesses and evaluation utilities
- run capture, analysis, and score-tracking tools
- agent workflows, prompts, and decision policies
- model comparison where it directly informs benchmark performance
- challenge-specific strategy documents for current or upcoming BitGN rounds

## Out of scope until explicitly added

- generic LLM experimentation without a benchmark-driven reason
- production infrastructure unrelated to building or evaluating the benchmark agent
- undocumented side projects that do not contribute to agent quality or evaluation speed
- assumptions about BitGN challenge behavior that are not written down or verified

## Scope notes

Update this file whenever the repository takes on a materially new responsibility.
