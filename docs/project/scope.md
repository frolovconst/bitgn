# Scope

## In scope

- BitGN-oriented agent development.
- Modular agent experimentation (strategy, prompting, memory, loop).
- Benchmark integration support and run workflows.
- Workspace-based reproducible local development.
- Minimal experiment records: commit SHA + benchmark score.

## Out of scope (until added)

- Generic LLM work not tied to benchmark results.
- Production infrastructure goals.
- Large undocumented side projects.
