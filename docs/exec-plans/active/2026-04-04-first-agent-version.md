# First Agent Version

## Goal

Implement the first benchmark-facing agent that can attempt the full BitGN task set through the platform layer.

## Scope

In scope:

- first end-to-end agent loop
- use of the minimum benchmark platform
- simple but complete task-handling behavior

Out of scope:

- advanced planning or memory systems before first full runs
- major optimization before baseline capability exists

## Steps

1. Define the minimum agent loop that can process benchmark tasks end to end.
2. Implement the first agent on top of the platform layer.
3. Run the agent against the full task set and establish the first baseline score.

Not started

## Notes

- Keep the first version simple.
- If implementation expands significantly, split follow-up work into smaller plans.
